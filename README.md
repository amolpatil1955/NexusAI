# 🚀 NexusAI – AI Resume Builder, ATS Checker & Smart AI Chat

![NexusAI Banner](./assets/banner.png)

> Experience AI-powered resume building with conversational intelligence.

NexusAI is a full-stack AI-powered platform that combines a **ChatGPT-like assistant**, **Resume Builder**, and **ATS Checker** into a single application.  
It helps users create professional resumes, optimize them for ATS systems, and interact with AI for career guidance.

---

## ✨ Features

### 🤖 AI Chat Assistant
- ChatGPT-like real-time chat interface
- Ask career-related questions
- Generate summaries, skills, and content
- Resume improvement suggestions

### 🧾 AI Resume Builder
- Create resumes using AI
- Edit and customize easily
- Professional structure and formatting

### 📊 ATS Checker
- Analyze resume ATS score
- Keyword optimization suggestions
- Improve selection chances

### 📈 Dashboard
- Manage all resumes
- Track ATS scores
- User profile management

### 🔐 Authentication
- Secure Login & Signup
- JWT-based authentication
