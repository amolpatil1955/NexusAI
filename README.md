# 🚀 NexusAI – AI Resume Builder & ATS Checker

![NexusAI Banner](./assets/banner.png)

NexusAI is an AI-powered Resume Builder and ATS Checker that helps users create professional resumes and optimize them for Applicant Tracking Systems (ATS).

---

## ✨ Features

- 🤖 AI Resume Builder
- 📊 ATS Score Checker
- 🧾 Resume Management (Create/Edit/Delete)
- 📄 Download Resume as PDF
- 🔐 Authentication System
- 📈 Dashboard with analytics

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Tailwind CSS / ShadCN UI

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**AI**
- OpenAI API / Gemini API
